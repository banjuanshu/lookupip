# lookupip
写lookupip是因为我的机器上的Python环境升级了到3.5.0, 由于Python3为了轻装上阵,没有做向下兼容.所以重写此workflow.
查询ip的地址位置,alfred3 是非常好的工具,可以大大提高工作效率.


# 数据源
基于淘宝ip地址库的ip查询

# 安装
直接下载Lookup IP.alfredworkflow,双击即可安装.
